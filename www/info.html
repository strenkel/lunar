LUNAR LANDER SIMULATOR
----------------------
 
THE LUNAR LANDING GAME WAS WRITTEN BY JIM STORER IN 1969. IT WAS CODED IN FOCAL FOR A DEC PDP-8 MINICOMPUTER. IT BECAME ONE OF THE MOST INFLUENTIAL COMPUTER GAMES EVER. THE LUNAR LANDER SIMULATOR IS AN ACCURATE RECODING IN JAVASCRIPT AND HTML OF THE FOCAL CODE FROM 1969.
